# Stagger Tracker

Patched version of [Stagger Tracker](https://www.esoui.com/downloads/info2555-StaggerTracker.html) that supports subclassing.

It replaces the DK class check by Earthen Heart skill line check. It also supports subclass change without having to reload.

## Changelog

1.1: Update 49 - Track new Heat Shock debuff instead of old Stagger.

1.0: Subclassing Support - Check skill line instead of base class.
